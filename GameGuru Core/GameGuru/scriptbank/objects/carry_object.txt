Carry Object 

Use add this script to carry a selected object.
Apply to an object that you wish to be carried
Set Object to Physics=ON, Gravity=ON

Set the behaviours: 
Set the PICKUP TEXT when in range
Set the PICKUP RANGE when object can be picked up, if left 
blank or too low it will auto compensate for object size.
Set the RELEASE TEXT instruction for releasing the object
Set REARM WEAPON to auto re-arm your weapon after object drop 
<Sound0> for pickup sound

Pickup with E or LMB or use RMB if want to throw object
While carrying Object MouseWheel to adjust carry height
